# Sprint 20 Validation Readiness Dashboard

**Royal Canin | Release 5 - Sprint 20 | Order Management Workstream**

## 📊 Dashboard Overview

This dashboard tracks validation readiness for Sprint 20 user stories with real-time countdown to the validation deadline (July 16, 6:00 PM CEST).

### Features
- **Real-time Countdown**: Live countdown timer to validation deadline
- **KPI Cards**: At-a-glance metrics for validation progress
- **Interactive Charts**: Visual representation of validation status
- **Detailed Tracker**: Sortable, filterable table of all user stories
- **Status Heatmap**: Quick visual scan of validation completion
- **Alert System**: Highlights stories requiring immediate attention

### Validation Types Tracked
- UX Validation
- Data Validation
- Business Validation
- Solution Validation

## 🚀 Quick Start

Simply open `index.html` in any modern web browser. No server or dependencies required.

## 📈 Data Source

Dashboard data is extracted from Azure DevOps (ADO) and refreshed periodically. Last extraction timestamp is displayed in the dashboard header.

## 🌐 GitHub Pages Deployment

This repository is configured for GitHub Pages deployment. Access the live dashboard at:

**https://[your-username].github.io/sprint-validation-dashboard/**

## 📝 Updates

To update the dashboard data, edit the JavaScript data objects in `index.html`:
- `sprintScope`: Expected validations for each story
- `adoData`: Actual validation status from ADO
